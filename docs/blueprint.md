# **App Name**: CET Prep Pro

## Core Features:

- Admin Test Upload: Upload test questions, options, and solutions via JSON files for Physics, Chemistry, and Mathematics sections.
- Realistic Exam Simulation: Simulate the MHT CET exam environment with accurate timing (180 minutes), section-wise question access, and auto-submission.
- Section Timers: Display clear, persistent timers for each 90-minute section, ensuring students adhere to the allocated time.
- Automatic Save: Automatically save student's selected answers every 2 minutes to prevent loss of progress, even with disconnection, while in the live exam. Provide ability to resume the test upon re-connection.
- Detailed Analytics Dashboard: Provide overall and subject-wise performance analysis, including accuracy, time management, and graphical representation of time spent per question.
- Question categorization and validation: Use an AI tool to evaluate the test json and provide error reporting and detailed next steps so the uploaded tests conform to exam rules, such as question format and expected solution format.
- Test History & Review: Allow students to access completed tests from the "History" tab and review the question, the answer they marked, the correct answer, and the detailed solution for each question.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) to convey professionalism and trust.
- Background color: Light gray (#F0F4F8), providing a neutral, clean backdrop.
- Accent color: Vibrant orange (#FF9100) for CTAs and important information.
- Body font: 'Inter' (sans-serif) for clean readability.
- Headline font: 'Space Grotesk' (sans-serif) to give a computerized, techy feel.
- Use clear, intuitive icons for navigation and section differentiation.
- Design a clean, professional interface mimicking the actual MHT CET exam environment. Incorporate a question palette for easy navigation.
- Subtle transitions and loading animations to enhance user experience without distraction.
